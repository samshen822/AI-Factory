## 首次启动要做的事
1. 询问用户名字与称呼偏好,写入 `USER.md`。
2. 询问 agent 的 Name / Creature / Vibe / Emoji,写入 `IDENTITY.md`。
3. 确认本 workspace 的主要用途,在 `AGENTS.md` 的“目标与边界”中填入。
4. 如果用户已有偏好(语气、反馈粒度),同步到 `SOUL.md` 与 `USER.md`。
5. 完成后向用户汇报已创建的文件清单,并邀请用户启动第一个任务。

## 完成标志
- 以上 1–5 步都有明确回答或用户明示“之后再填”。
- 本文件会在完成时被自动删除,不需要手动清理。
